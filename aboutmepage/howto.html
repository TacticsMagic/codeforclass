How I made this webpage

What HTML did I use?

I have learned to use the <p></p> tags in order to write out the paragraphs that you see here. The <h1></h1> tags were used to create the headers you see at the top of this webpage. <a href=""></a> tags were used to provide links to the About Me page and How To page. The <ul><li></li></ul> tags are used to create unordered lists and while <ol></ol> makes ordered lists. I used the unordered lists in order to organize the things that I like because from a design perspective it does not give. The <img src=""> tag is used to put in the images that you see on the about me webpage. I used the <div></div> tags to divide up the code in order to style them with CSS. I used the <br> tag to create space between two elements to keep the page from being all compressed.

What CSS did I use?

I was able to style the HTML by linking it with the external stylesheet with the following code <link rel="stylesheet" href="aboutme.css" type="text/css"/> The navigation bar was styled by selecting .navbar and styled it with background color, borders, and even a sticky position in order to give it more flair. When the mouse hovers over the navbar links I made it change from blue to orange with the selector .navbar a:hover. I was able to align the text to the center by using the body selector and used text-align: center; to center the text. I used the font-size property to make the text bigger and more legible. I was able to put two pictures side by side by making .line as my selector and used display: inline-block; to put two pictures side by side to make it look neater as you can see down below. I used the border property to separate elements from each other and make it neater.

How to make a navbar

The navbar in its simplest form is basically an unordered list with links in the text

But in order to make it look like an actual navbar you will need CSS to style it. If you want the options to be side by side you must use the property inline-block then you can use properties such as border to give the navbar borders or color to change the color of the text. If you really want to get advanced, you can select a:hover to make the navbar change color once the mouse hovers over the links.
